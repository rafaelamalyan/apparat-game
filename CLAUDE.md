# CLAUDE.md — контекст проекта «Аппарат»

Это файл-память для Claude Code. Читай его перед работой над проектом.

## Что это
2D-аркада на **Phaser 3** + **Vite**. Игрок — Серёга (руководитель Аппарата) —
ловит поручения, которые кидает Карен (генеральный директор), и сгружает их
в лоток нужного департамента по цвету. Тема — сатира на аппаратную работу.

## Как запустить
```bash
npm install
npm run dev      # запуск dev-сервера с горячей перезагрузкой (порт 5173)
npm run build    # сборка в dist/
npm run preview  # предпросмотр сборки
```

## Структура
```
index.html              — точка входа Vite
src/
  main.js               — конфигурация Phaser, список сцен
  core/
    config.js           — ВСЕ константы: размеры, палитра PAL, департаменты DEPTS, баланс BALANCE
    office.js           — процедурный фон-кабинет (окно, город, кран, свет)
  scenes/
    BootScene.js        — загрузка спрайтов + генерация текстур (папки, частицы, виньетка)
    MenuScene.js        — стартовый экран
    GameScene.js        — основной игровой цикл (главный файл логики)
    OverScene.js        — экран окончания
public/
  sprites/              — PNG персонажей (karen_*, sergey_*)
```

## Где что менять
- **Баланс сложности** (скорость, темп, жизни, руки) — только `src/core/config.js`, объект `BALANCE`.
- **Цвета и палитра** — `src/core/config.js`, объекты `PAL` и `DEPTS`.
- **Игровая логика** (ловля, маршрутизация, настроение) — `src/scenes/GameScene.js`.
- **Фон/атмосфера** — `src/core/office.js`.

## Персонажи (спрайты)
- `karen_idle / karen_throw / karen_angry` — генеральный: покой, бросок, злой бросок.
- `sergey_idle / sergey_catch / sergey_carry` — Аппарат: покой, ловит (руки раскрыты), несёт папки.
- Спрайты сгенерированы во внешнем генераторе по фото реальных людей (дружеский шарж),
  фон снят программно. При замене — класть PNG с прозрачным фоном в `public/sprites/`.

## Текущее состояние
- Реализовано: ловля, 3 департамента, цветные папки, серые поручения, таймеры,
  настроение босса с реакцией мира (краснеет), комбо, частицы, тряска экрана.
- Department-лотки перегружаются при 4 сгрузках подряд (блок на ~4 сек).
- **Мобильная версия:** адаптивный масштаб (`Phaser.Scale.FIT` в `main.js`),
  сенсорное управление в `GameScene.setupTouch()` — вести пальцем (движение
  за пальцем своей скоростью) + тап по лотку для сгрузки. В портретной
  ориентации `index.html` показывает подсказку «поверни телефон».
- Опубликовано на GitHub Pages, авто-деплой через `.github/workflows/deploy.yml`.
