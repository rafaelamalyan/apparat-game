# Аппарат · поток поручений

2D-аркада на Phaser 3. Серёга (руководитель Аппарата) ловит поручения генерального
директора Карена и сгружает их в нужный департамент по цвету папки.

![управление](https://img.shields.io/badge/управление-стрелки%20%2F%20A%20D%20%2F%20пробел-e8b04a)

## Запуск

```bash
npm install
npm run dev
```

Откроется `http://localhost:5173`.

## Управление
- **← →** или **A / D** — двигать Серёгу
- **↓** или **пробел** над лотком — сгрузить поручение в департамент того же цвета

## Правила
- Карен кидает поручения трёх цветов (Проект — синий, Стройка — оранжевый, Финансы — зелёный).
- Поймай и сгрузи в лоток того же цвета: верно — очки и серия, мимо — штраф и злость босса.
- Серое поручение со знаком «?» примет любой департамент, но без бонуса.
- В руках максимум 4 поручения. Просрочка (таймер по ободку) или падение — минус жизнь.
- Чем злее босс, тем быстрее темп и сильнее краснеет экран.
