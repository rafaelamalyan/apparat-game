# Режим «АППАРАТНАЯ ДУЭЛЬ» — дизайн-док

Файтинг в стиле Mortal Kombat, где бьются не кулаками, а **аппаратными приёмами**.
Это новый игровой режим внутри игры «Аппарат · поток поручений». Документ — источник
правды по фиче (обновлять по ходу).

## Концепт
- **Стейдж:** кабинет генерального (фон `public/bg/office.jpg`, уже есть).
- **Карен Гаспарович (Шеф) — судья/анонсер** на троне-кресле: «ФАЙТ!», «ЗАДВИНУТЬ ЕГО!»
  (роль Шао Кана в МК).
- **Бойцы:** **Серёга** (игрок) vs **«Проверка СВА»** (соперник — олицетворённая
  служба внутреннего аудита; на след. дуэлях — другие соперники, лестница).
- **Здоровье = «ПОЗИЦИИ»** (аппаратный вес). Ноль → «ЗАДВИНУТ / УВОЛЕН».
- **Супер-метр «АДМИНРЕСУРС»** копится от ударов; полный → ультимейт.

## Соперник «Проверка СВА»
- **Женщина-ревизор**, олицетворение внутреннего аудита: строгая, 35–45 лет, тёмный
  деловой костюм, очки, волосы собраны, портфель с актами/предписаниями. Холодная,
  методичная, начальственная (антагонист).
- **Его приёмы (зеркало аппаратных):** «Замечание» (лёгкий), «Запрос документов»
  (дальний), «Акт проверки» (средний), «Предписание» (тяжёлый), «Финансовое нарушение»
  (тяжёлый), «Передать материалы в правоохранительные органы» (ульта).

кинематографичная.

## Механика
- **Бой до 2 побед (best of 3)**, таймер раунда ~60с; по таймауту — у кого больше «Позиций».
- **Блок** режет урон. **Метр** копится от нанесённых/полученных ударов.
- **Дальние** приёмы (записка, докладная) бьют через экран; ближние — в упор (нужна дистанция).
- **ИИ соперника:** подходит, бьёт, иногда блокирует, к концу наглеет/ускоряется.
- **Сочность (как МК):** хитспарки, тряска экрана, hitstop (заморозка кадра в момент
  удара), комбо-счётчик, голос-анонсер, «ФАЙТ!»/«K.O.».
- **Победа → промпт «ЗАДВИНУТЬ ЕГО!»** → добивание (фаталити-сценка).

## Управление
- Десктоп: ← → ходить · **J** лёгкий · **K** средний · **L** тяжёлый · **U** спец/ульта ·
  **S/пробел** блок. Направление+кнопка = именной приём (→+L = «Срезать премию» и т.п.).
- Телефон: экранные кнопки (ходьба слева, удары/блок справа).

## Встройка в карьеру
- Раунд 3 (и далее каждый N-й) = **дуэль** вместо ловли. Победа → крупная премия +
  «повышение»; поражение → теряешь позиции/жизнь. Лестница соперников всё круче.
- Общая петля сохраняется: раунд → премия → магазин.

## План реализации (фазы)
1. ✅ Дизайн-док (этот файл).
2. **Арт** (в ChatGPT): позы бойцов + эффекты. ⚠️ нужны НОВЫЕ позы **вид сбоку/3-4**
   (текущие спрайты анфас — для файтинга не годятся).
3. **Каркас боя:** сцена, 2 бойца, полоски «Позиции», ходьба, лёгкий/средний/тяжёлый,
   попадания, блок, KO, ИИ. Анимация поза-в-позу через твины + эффект приёма.
4. **Сочность:** спарки, тряска, hitstop, комбо, callouts.
5. **Метр + ульта ДЭБ + добивание.**
6. **Встройка в карьеру + баланс + мобильные кнопки.**

## Чеклист арта (на каждого бойца), единый стиль, вид сбоку (3/4), прозрачный фон
1. стойка (idle) 2. шаг 3. лёгкий удар 4. средний удар 5. тяжёлый замах + удар
6. спец-приём 7. получает удар (стаггер) 8. блок 9. победа 10. нокаут (лицом в стол).

**Эффекты-предметы:** бумажка-записка, печать «НЕ СОГЛАСОВАНО», конверт-докладная,
ножницы+₽, «ДЭБ» (силуэты людей в форме). Для «Проверки СВА» — акт/предписание, штамп
«НАРУШЕНИЕ».

## Промты для генерации арта (ChatGPT)
Для единства стиля к ОБОИМ промтам прикладывать существующий спрайт/скриншот.
Позы v1 (8): стойка, лёгкий удар, тяжёлый удар, спец-приём, получает удар, блок,
победа, нокаут. Все фигуры одного размера, на одной линии пола, прозрачный фон.
Серёга — лицом вправо; «Проверка СВА» — лицом влево (направление некритично, флипается в коде).
Соперник — женщина-ревизор. Полные тексты промтов выданы в чате; props: записка, штамп
«НЕ СОГЛАСОВАНО», конверт-докладная, ножницы+₽, штамп «ВЫГОВОР», силуэты «ДЭБ» (маски-шоу),
для СВА — «АКТ ПРОВЕРКИ» и штамп «НАРУШЕНИЕ».
